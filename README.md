# shadcn for ImGui

A shadcn-inspired utilities for Dear ImGui.

## About

This project originally started as an attempt to bring the clean, polished, and well-structured design of shadcn/ui into the world of Dear ImGui.

The goal was simple: beautiful and clean. But ImGui's very hard to work with due to it's shitty structure, and most % of times it doesn't even work correctly.

A lot of things feel hacked together over time. The architecture isn’t always intuitive, and extending it cleanly can turn into a fight. Building a higher-level design system on top of it is harder than it probably should be.

That said, this project exists because I still believe something better can be built on top of it.

## Current State

Right now this library is:

- Experimental
- Likely unoptimized
- Possibly unstable

I will try to maintain and improve it, but it needs some work.
